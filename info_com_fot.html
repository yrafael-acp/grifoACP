Recepción de Combustible
Inicie sesión con Google para acceder. Los informes se guardan en Google Drive y son visibles para todos los usuarios con acceso a la carpeta.
⚙️ Configuración requerida (una sola vez — solo el administrador):

1. Ir a console.cloud.google.com → Crear proyecto
2. Activar Google Drive API y Google Identity
3. Crear credencial OAuth → tipo Aplicación web
4. Agregar file:// o la URL donde aloje este HTML como origen autorizado
5. Copiar el Client ID y reemplazar TU_CLIENT_ID_AQUI en el script
